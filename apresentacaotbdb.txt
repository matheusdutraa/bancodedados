1- Como faz para importar e exportar?

Para importar utilizamos os comandos:

mysql -u root -p"senha" "nomeDaDatabaseQueVoceQuerEnviarOarquivo" < "nomeDoCaminhoParaOarquivo+nomeArquivo"

Para exportar é parecido, fazemos:

mysqldump -u root -p"senha" > "caminhoDoArquivo+nomeAficarOarquivoExportado"
